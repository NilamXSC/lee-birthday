# Assets Directory

This folder contains all the media assets for Leena's birthday webpage.

## Required Audio Files

Please place the following audio files in this directory:

### 1. `happy-birthday.mp3`
- **Purpose**: Background music that plays after candles are blown
- **Recommended**: Soft, romantic instrumental version of "Happy Birthday"
- **Format**: MP3
- **Note**: The audio will loop automatically

### 2. `blow-sound.mp3` (Optional)
- **Purpose**: Sound effect when blowing out candles
- **Recommended**: Soft "whoosh" or wind sound
- **Format**: MP3
- **Note**: If not provided, the page will work without this sound

### 3. `chime.mp3` (Optional)
- **Purpose**: Celebratory chime when candles are blown
- **Recommended**: Gentle bell or chime sound
- **Format**: MP3
- **Note**: If not provided, the page will work without this sound

## Audio Source Suggestions

You can find free, royalty-free audio from:
- [Freesound.org](https://freesound.org)
- [Pixabay Music](https://pixabay.com/music/)
- [Incompetech](https://incompetech.com/music/) (Kevin MacLeod)
- [Bensound](https://www.bensound.com)

## Image Assets (Optional)

If you want to add custom images, you can create:
- `background-pattern.png` - Custom background texture
- `decorative-elements.png` - Additional decorative graphics

**Note**: The current design works entirely with CSS and emojis, so images are optional.

## Customization

To use different audio files:
1. Replace the files in this directory with your own
2. Update the file names in `index.html` if using different names
3. Ensure audio files are in MP3 format for best browser compatibility
